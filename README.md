# PicSeek
PicSeek is an image search tool similar to Samsung’s Circle To Search.

# Support
PicSeek is written in Python and it offically supports almost all Ubuntu / Debian based distros.

# Note
This is a beta release. It is only released for testing purpose. It may not properly work on diffrent desktop enviroments, window managers and display servers. This version was tested on Linux Mint 21.1 Vera with Cinnamon 5.6.8 . This version will not work on Wayland for sure.
